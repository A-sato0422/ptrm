# トレーナー管理画面 - 実装ガイド

## 📋 実装済み画面

### ✅ 顧客一覧画面 (`clients.html`)

トレーナーが担当する顧客を一覧で確認・管理する画面です。

#### 主な機能

1. **顧客情報の表示**
   - 顧客の基本情報（名前、ID、アバター）
   - 最終更新日とメモ
   - 4つのカテゴリー別レベル表示（Blue Pilates、Red Weight、Green Sports、Yellow Movement）

2. **検索機能**
   - 顧客名またはIDで検索可能
   - リアルタイムフィルタリング

3. **タスク管理**
   - 各顧客の現在の課題をチェックボックスで管理
   - タスクの完了/未完了を切り替え可能

4. **顧客の嗜好表示**
   - やりたいトレーニング
   - やりたくないトレーニング

#### アクセス方法

開発サーバーを起動後、以下のURLにアクセス：
```
http://localhost:5174/clients.html
```

#### 技術スタック

- **UI**: Tailwind CSS（CDN版）
- **フォント**: Inter + Noto Sans JP
- **アイコン**: Material Icons Outlined
- **ロジック**: TypeScript（`src/clients.ts`）

## 🚀 開発サーバーの起動

```bash
npm run dev
```

起動後、`http://localhost:5174/clients.html` にアクセスしてください。
