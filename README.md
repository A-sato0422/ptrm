# PTRM - パーソナル・トレーニング・ロードマップ

パーソナルジムのクライアント向けモチベーション管理システム

## 🎯 概要

PTRMは、パーソナルジムのクライアントが自らの成長を可視化し、トレーナーとの間で「フィードバック→確認→挑戦」のサイクルを形成するためのシステムです。

## 🛠️ 技術スタック

- **フロントエンド**: TypeScript
- **ビルドツール**: Vite
- **バックエンド**: Supabase (BaaS)
- **ホスティング**: Vercel
- **認証**: LINE LIFF (LINE Front-end Framework)

## 📱 対応環境

- クライアント画面: スマートフォンのみ対応
- トレーナー管理画面: PC + スマートフォン対応
- PWA対応（ホーム画面への追加可能）

## 🎨 デザインコンセプト

- 手書きチョークアート風
- パステルカラー
- メインカラー: 水色 (#A8D8EA)

## 🚀 開発環境のセットアップ

### 前提条件

- Node.js (v18以上推奨)
- npm または yarn

### インストール

```bash
# リポジトリのクローン
git clone <repository-url>
cd dev-PTRM

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

### ビルド

```bash
# プロダクションビルド
npm run build

# ビルドのプレビュー
npm run preview
```

## 📄 画面一覧

### クライアント側画面（P0機能）

1. **ダッシュボード（ホーム画面）** - `index.html`
   - 4項目のレベル表示（山のビジュアライゼーション）
   - アクションリストプレビュー
   - 次のチャレンジ項目
   - 好み評価の確認

2. **アクションリスト画面** - `action.html`
   - カテゴリー別のタスク一覧
   - タスクの完了/未完了管理
   - 好み評価（Like/Dislike/どちらでもない）
   - 動画ページへのリンク

3. **6ステージ解説画面** - `stage.html`
   - 6つのステージの詳細説明
   - 各ステージのレベル範囲
   - 進捗状況の可視化
   - カテゴリー別のレベル表示

4. **プロフィール画面** - `profile.html`
   - ユーザー情報表示
   - 統計情報（平均レベル、セッション数等）
   - カテゴリー別レベル進捗
   - 達成バッジ表示
